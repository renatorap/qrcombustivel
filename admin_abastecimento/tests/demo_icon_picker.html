Características

  • Interface visual intuitiva e moderna
  • Busca em tempo real por nome do ícone

  • Grid responsivo com 100+ ícones

  • Compatível com múltiplos campos na mesma página
  • Efeitos hover e animações suaves
  • Bootstrap 5 e FontAwesome 6

Exemplo de Código

Como usar o Icon Picker em seu formulário:

<!-- 1. Incluir o script -->
<script src="../js/icon_picker.js"></script>

<!-- 2. HTML do campo -->
<div class="input-group">
  <span class="input-group-text" id="iconPreview">
    <i class="fas fa-home"></i>
  </span>
  <input type="text" id="iconInput" value="fa-home">
</div>

<!-- 3. Inicializar -->
<script>
  makeIconInputClickable('iconInput', 'iconPreview');
</script>
